# Connect

Connect your dapp to Algorand users's wallets.
User should grant access and choose the wallets that wish to share with your dapp.

### Preconditions

* User must have MyAlgo Wallet Account previously set in **[MyAlgo Wallet](https://wallet.myalgo.com/home)** to be able to connect to the dapp. Otherwise user will be alerted in MyAlgo Connect popup.

* User should have a wallet previously added in **[MyAlgo Wallet](https://wallet.myalgo.com/home)** in order to be able to share it. Otherwise user will be aleted in MyAlgo Connect popup.
